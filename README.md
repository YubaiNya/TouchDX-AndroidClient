# TouchDX for Android

**TouchDX for Android** 是基于MaiMaiDX的街机原生触控思路开发的安卓端串流触摸客户端，旨在解决使用串流工具游玩时，傻逼Windows手势干扰游玩的过程

## 功能特性

*   **模拟官机的触摸方案**: 采用了类似于官机的区块触摸方案，独立与电脑程序通信，不会受到傻逼手势的干扰
*   **高精度、低延迟的触摸模拟**: 采用优化的网络协议和触摸事件处理机制，确保您的每一次点击、滑动都能被精准、迅速地传递到 PC 端，带来媲美物理按键的响应速度。
*   **多点触控支持**: 完美支持多点同时触摸，轻松应对需要复杂手势和多指操作的游戏场景。
*   **轻量化与低资源占用**: 客户端本身经过精心优化，确保在提供强大功能的同时，尽可能地减少对您安卓设备性能的消耗。

## 使用教程

### 准备工作

1.  **安装 TouchDX PC 端 Mod**: 确保您已经在您的 PC 上，为您想要玩的游戏正确安装了 TouchDX 的 PC 端 Mod。
2.  **网络环境选择**: 
    *   **【推荐】有线连接 (ADB Reverse Forwarding)**: 为了获得最佳的低延迟体验，我们强烈建议您通过 USB 数据线连接您的安卓设备和 PC，并使用 ADB (Android Debug Bridge) 的反向代理功能。这可以为您提供最稳定、最快速的连接。
    *   **无线连接 (Wi-Fi)**: 将您的安卓设备和 PC 连接到同一个局域网（Wi-Fi）下。这是一个更方便的选择，但可能会受到网络环境波动的影响。
3.  **获取 PC 的 IP 地址 (仅限 Wi-Fi 连接)**: 在您的 PC 上，通过 `PCDiagnosticTool` 或其他方式，获取您的局域网 IP 地址（通常以 `192.168.` 开头）。

### 安装与配置客户端

1. **下载并安装**: 从本项目的 [Releases](https://github.com/TouchDX/TouchDX-AndroidClient/releases) 页面，下载最新的 `.apk` 安装包，并将其安装到您的安卓设备上。

2. **【推荐】配置 ADB 有线连接**: 

   * 在您的 PC 上，确保已经安装了 [Android SDK Platform Tools](https://developer.android.com/studio/releases/platform-tools)。

   * 使用 USB 数据线连接您的安卓设备和 PC，并确保已经开启了“USB 调试”模式。

   * 在 PC 上打开一个终端（命令行或 PowerShell），然后输入以下命令：

     ```
     adb reverse tcp:4321 tcp:4321
     ```

   * 这条命令会将您 PC 上的 4321 端口，转发到您安卓设备上的 4321 端口。

3. **打开应用并配置**: 

   *   打开您设备上的 “TouchDX” 应用。
   *   **如果您使用了 ADB 有线连接**，请在 IP 地址输入框中，输入 `127.0.0.1` (代表本机)。
   *   **如果您使用 Wi-Fi 连接**，请在 IP 地址输入框中，填入您 PC 的局域网 IP 地址。
   *   点击“连接”按钮。

4. **确认连接状态**: 

   *   如果一切顺利，屏幕上会显示“已连接”的提示。
   *   同时，您可以在 PC 端的 `PCDiagnosticTool` 中看到来自您设备的连接，并且您在屏幕上的触摸操作会实时反映在诊断工具的界面上。

### 开始游戏！

1.  在 PC 上启动已经安装了 TouchDX Mod 的游戏。
2.  现在，您在安卓设备上的所有触摸操作，都将被实时地模拟为游戏内的按键和触摸事件。
3.  尽情享受 TouchDX 为您带来的全新游戏体验吧！

## 常见问题 (FAQ)

**Q: 连接失败，或者连接后没有反应怎么办？**

A: 请按照以下步骤进行排查：

      1.  **检查网络/USB连接**: 
          *   **Wi-Fi 用户**: 确保您的手机和 PC 在同一个 Wi-Fi 网络下。可以尝试互相 `ping` 一下对方的 IP 地址，检查网络是否通畅。
          *   **ADB 用户**: 确保您的 USB 数据线连接稳定，并且在终端中输入 `adb devices` 可以看到您的设备处于 `device` 状态。
      2.  **检查 IP 地址**: 
          *   **Wi-Fi 用户**: 确认您在客户端中输入的 IP 地址，与 PC 端 `PCDiagnosticTool` 中显示的 IP 地址完全一致。
          *   **ADB 用户**: 确认您在客户端中输入的 IP 地址是 `127.0.0.1`。
      3.  **检查防火墙**: 检查您 PC 上的防火墙（Windows Defender 防火墙或其他安全软件）是否阻止了游戏的网络连接，或者阻止了 TouchDX 的通信端口 (默认为 4321)。
      4.  **检查 Mod 是否加载**: 确认 PC 游戏中的 Mod 是否已经成功加载。

**Q: 延迟很高，操作不跟手怎么办？(主要针对 Wi-Fi 用户)**

A: **我们强烈推荐您使用 ADB 有线连接方式来获得最佳体验。** 如果您必须使用 Wi-Fi，可以尝试以下方法来缓解延迟：

      1.  **使用 5GHz Wi-Fi**: 相比 2.4GHz Wi-Fi，5GHz 通常拥有更高的速度和更低的干扰。
      2.  **靠近您的路由器**: 减少手机、PC 与路由器之间的物理距离和障碍物。
      3.  **关闭其他占用带宽的应用**: 在游戏时，尽量关闭手机和 PC 上其他正在进行下载、视频播放等占用大量网络带宽的程序。
